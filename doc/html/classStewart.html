Detailed Description

Stewart Controller Class This class takes care of interfacing with Webots control API and implementing kinematics and trajectory plan for the Stewart Platform.

Kinematics are implemented according to "Kinematic and dynamic analysis of Stewart platform-based machine tool structures" by Khalifa Harib and Krishnaswamy Srinivasan [1]

Author
Fabio Polisano
